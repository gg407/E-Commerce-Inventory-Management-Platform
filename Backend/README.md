# ecommerce-platform (backend)

Spring Boot 3.2 / Java 21 REST API for the e-commerce platform. See the [project root README](../README.md)
for full setup instructions (Docker, manual setup, seed accounts, API summary).

Quick start:

```bash
./mvnw spring-boot:run
```

Run with a specific profile:

```bash
./mvnw spring-boot:run -Dspring-boot.run.profiles=dev
```

Run tests:

```bash
./mvnw test
```
